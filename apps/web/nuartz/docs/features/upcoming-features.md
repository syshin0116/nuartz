These are features we're considering or actively working on for nuartz. This list is not a commitment — priorities may shift based on community feedback.

## Under consideration

- **Dead link detection** — Static analysis to find broken `[[wikilinks]]` at build time
- **Sidenotes** — Margin notes as an alternative to footnotes
- **Exact-match search** — Use double quotes in search for exact phrase matching
- **Canvas support** — Rendering Obsidian Canvas (`.canvas`) files
- **Obsidian URI support** — `obsidian://` protocol links for opening notes in Obsidian

## Want to help?

If you're interested in working on any of these, check out [[contributing]] to get started. Feature requests and ideas are welcome as GitHub issues.
