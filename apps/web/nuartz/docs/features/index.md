Nuartz includes a rich set of features for publishing and navigating your digital garden. Here's everything available:

## Writing & Syntax

- [[callouts]] — Obsidian-style callout blocks (`> [!note]`, `> [!warning]`, etc.)
- [[wikilinks]] — Link between notes with `[[double brackets]]`
- [[tags]] — Inline `#tags` and frontmatter tags
- [[syntax-highlighting]] — Code blocks with dual-theme highlighting
- [[latex]] — Math equations with KaTeX
- [[mermaid]] — Diagrams and flowcharts
- [[comments]] — Reader comments via Giscus (GitHub Discussions)

## Navigation & Discovery

- [[graph-view]] — Interactive graph of note connections
- [[backlinks]] — See which pages link to the current page
- [[search]] — Full-text search with `Cmd+K` command palette
- [[table-of-contents]] — Auto-generated heading navigation
- [[breadcrumbs]] — Path-based breadcrumb navigation
- [[explorer]] — File tree browser in the sidebar
- [[folder-and-tag-listings]] — Auto-generated listing pages

## Reading Experience

- [[dark-mode]] — Light and dark theme with system preference detection
- [[popover-previews]] — Hover previews for internal links
- [[reader-mode]] — Distraction-free reading
- [[obsidian-compatibility]] — Full compatibility with Obsidian vault content

## Publishing

- [[private-pages]] — Draft and private page support
- [[rss-feed]] — RSS feed generation
- [[social-images]] — Open Graph image generation
- [[spa-routing]] — Client-side navigation

## Planned

- [[citations]] — Academic citation support (BibTeX)
- [[i18n]] — Internationalization support
- [[upcoming-features]] — Other features on the Nuartz roadmap
